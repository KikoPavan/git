Configuração de usuário, email e do editor.
FLUXO DE TRABALHO
Seus repositórios locais consistem em três "árvores" mantidas pelo git. 
A primeira delas é sua Working Directory que contém os arquivos vigentes. 
A segunda Index que funciona como uma área temporária.
E finalmente a HEAD que aponta para o último commit (confirmação) que você fez.
Configurando o Git
-Configurar usuário "git config --global user.name "Kiko Pavan"
-Configurar email   "git config --global user.email "kikopavan@hotmail.com"
-Configurar editor “git config --global core.editor "caminho do editor"
3 - 📂 Curso GIT e GITHUB - Comandos básicos
Duração 12:09
Criando repositório local 
- Criar um repositório local "git init"
- Verificar o estado do Container "git status"
- Criar um arquivo dentro do diretório “touch exemplo.txt”
- Adicionar o arquivo ou os arquivos ao container “git add exemplo.txt” todos “git add *”
- Verifica o container “git status” e envia para o repositório identificando a operação,
  com o comentário "git commit -m “comentário entre aspas”
- Comando para adicionar mudança de vários arquivos “git add *”
